# Tailwind Rules

- Use Tailwind CSS v4 with `@theme` directive
- Use CSS variables for theming (`--color-primary`, etc.)
- Use `cn()` utility for conditional classes
- Prefer utility classes over custom CSS
- Use `dark:` variant for dark mode
